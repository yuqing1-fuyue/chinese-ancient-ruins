# 中华古址智鉴 🏛️

> 全国重点城市古遗址 AI 科普系统（66 处遗址 · 11 座城市 · 3D 可视化）

---

## 目录结构

```
中华古址智鉴/
├── images/          ← 所有遗址图片（67 张，必须随项目一起复制）
├── main.py          ← FastAPI 后端服务（核心文件）
├── index.html       ← 主页面
├── map3d.html       ← 3D 地图页面
├── requirements.txt ← Python 依赖
├── 启动服务.bat     ← Windows 一键启动脚本
└── README.md        ← 本说明文件
```

---

## 迁移到其他电脑

**只需把整个 `中华古址智鉴` 文件夹（含 `images/` 子目录）完整复制过去即可。**  
图片路径采用相对路径设计，不依赖任何绝对路径，换电脑无需修改任何代码。

---

## 快速启动

### 方式一：双击 bat 脚本（Windows）

直接双击 `启动服务.bat`，脚本会自动检查 Python 环境、安装依赖并启动服务。

### 方式二：手动启动

```bash
# 1. 安装依赖（仅首次需要）
pip install -r requirements.txt

# 2. 启动服务
python main.py
```

### 访问地址

| 页面 | 地址 |
|------|------|
| 主页 | http://127.0.0.1:8080 |
| 3D地图 | http://127.0.0.1:8080/map3d |
| API文档 | http://127.0.0.1:8080/docs |

## 图片说明

- 项目共有 **67 张**遗址图片，存放在 `images/` 目录
- 图片格式：`.jpg` / `.jpeg` / `.png` / `.webp`
- 图片由 FastAPI 路由 `/images/{filename}` 动态提供
- 若某遗址无对应图片，系统会自动生成渐变色占位图

---

## 数据概览

- 覆盖城市：北京、西安、南京、洛阳、开封、杭州、承德、苏州、平遥、大同、扬州
- 遗址总数：66 处
- 时代跨度：旧石器时代 → 清代
- 世界文化遗产：多处
